# 🏅 Summer Olympics Dashboard

An interactive Power BI dashboard that provides insights into the performance, participation, and trends of the Summer Olympics over the years. This dashboard allows users to explore the evolution of the Olympic Games through clean visuals, KPIs, and historical data analysis.

## 📖 Project Overview

This Power BI project analyzes historical Summer Olympics data to understand:
- How different countries have performed over time
- Trends in athlete participation
- Medal distributions by gender and sport
- Overall evolution of the Olympic Games

This dashboard aims to simplify complex Olympic datasets into digestible insights for analysts, sports enthusiasts, and decision-makers.

---

## 🗂️ Dataset

- **Source**: [Kaggle - Olympic History Dataset](https://www.kaggle.com/datasets/heesoo37/120-years-of-olympic-history-athletes-and-results)
- **File**: `summer_olympics.csv`
- **Data Fields**:
  - Athlete name
  - Country (NOC)
  - Year
  - Sport
  - Event
  - Medal (Gold, Silver, Bronze)
  - Gender
  - Age

---

## ✨ Features

- Total medals won by each country
- Participation trends by year and gender
- Distribution of sports and events across years
- Top athletes by medal count
- Country-wise performance comparisons
- Drill-down capability and filters for detailed exploration

---

## 📊 Key Insights

- The USA has consistently been the top-performing country.
- Female participation has steadily increased over the decades.
- Athletics and Swimming are among the most medal-rich sports.
- Some countries dominate specific sports categories.
